# runlog.md 格式标准

## 条目标题格式

```markdown
## YYYY-MM-DD HH:MM — N条[（可选状态描述）]
```

- 日期时间使用 24 小时制，零填充（如 `08:03` 不是 `8:3`）
- N 为累计采集条数，使用千分位分隔（如 `11,000`）
- 状态描述可选：`崩溃`、`崩溃恢复`、`首次运行`

## 必填元数据字段

紧接标题后，按以下顺序：

```markdown
**运行时长**: [格式灵活，如 ~12小时、1h16m、约3.5小时]
**数据源**: [豆瓣读书 / 当当网 / 豆瓣读书+当当网]
**累计采集**: N / 100,000 (X%)
```

### 条件必填字段（有数据时必填）

```markdown
**当前速率**: ~N条/小时
**剩余**: N条
**预计完成**: YYYY-MM-DD ~ YYYY-MM-DD（约N天）
```

### 特殊情况字段（崩溃/错误时必填）

```markdown
**错误**: `完整错误消息`
**崩溃回退**: 描述丢失的数据量
```

## 分类分布表格

### 完整标准格式

```markdown
| 分类 | 已采集 | 目标 | 完成率 | 变化 |
|------|--------|------|--------|------|
| A 马列 | 1,118 | 1,500 | 75% | - |
| B 哲学宗教 | 2,650 | 5,000 | 53% | +1 |
```

- 22 大类全部列出，即使已采集为 0
- **排序规则**：按完成率降序 → 有进展的类在前 → 零进展类在后
- **变化列**：与上一时段的差值，0 时写 `-`，正值写 `+N`，首次运行时省略此列
- **完成率**：整数百分比，四舍五入
- **分类名**：简写中文名（如"马列"而非全称"马列主义、毛泽东思想、邓小平理论"）

### 进度条可视化格式（可选，用于日志粘贴）

```markdown
| 分类 | 进度条 | 已采集 | 目标 | 完成率 |
|------|--------|--------|------|--------|
| A | ##############...... | 1118 | 1500 | 75% |
```

- 进度条总长 20 字符
- 已完成为 `#`，未完成为 `.`

## 分析段

以 `### 分析` 或 `### 问题` 为标题，至少包含：

1. **最大变化类**：哪个分类突破/爆发，数值是多少
2. **停滞类**：哪些分类 delta=0，原因
3. **速率趋势**：相比上一时段是升是降，原因
4. （如有错误）**错误分析**：原因、影响、恢复步骤

格式为无序列表，每条 1-2 句，简洁务实。

## 状态标

```markdown
### 状态

运行中
```

或

```markdown
### 状态

已停止
```

## 条目分隔

每个条目之间用 `---`（水平线）分隔，水平线前后各一个空行。

## 示例（完整条目）

```markdown
## 2026-05-20 09:27 — 10000条

**运行时长**: ~13.5小时（5/19晚 ~ 5/20早）
**数据源**: 豆瓣读书
**累计采集**: 10,000 / 100,000 (10%)
**当前速率**: ~740条/小时
**预计完成**: 2026-05-25 下午（约5天后）

### 分类分布

| 分类 | 已采集 | 目标 | 完成率 | 变化 |
|------|--------|------|--------|------|
| A 马列 | 1,118 | 1,500 | 75% | - |
| B 哲学宗教 | 2,649 | 5,000 | 53% | +2 |
| E 军事 | 944 | 2,500 | 38% | +943 |
| Z 综合 | 5 | 4,000 | 0% | - |

### 备注

- E类（军事）本时段爆发增长: 1→944，自适应调度正在生效
- 其余冷门类仍接近0，标签可能已被标记为bad_tags

### 状态

运行中
```
