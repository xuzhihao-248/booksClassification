## 2026-05-20 13:14 — 崩溃: PermissionError

**累计采集**: ~12,000条（缓冲500条丢失，checkpoint在11000）
**错误**: `[Errno 13] Permission denied: books_100k.xlsx`

### 原因

Excel文件被外部程序打开锁定，`openpyxl` 保存时无法写入。

### 影响

- `_flush()` 抛异常 → 缓冲区内未写入的500条丢失
- `finally` 中 `close()` 再次尝试保存仍失败 → 程序完全退出
- checkpoint 最后写入在11000条，回退约1000条进度

### 恢复

```powershell
python main.py --resume
```

### 教训

**运行期间不要用Excel打开输出文件。** 用checkpoint查看进度：

```powershell
python -c "
import json
with open('data/checkpoint.json','r',encoding='utf-8') as f:
    d = json.load(f)
print(f'进度: {d[\"total_scraped\"]}/100000')
"
```
